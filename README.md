# Unity Slot Machine Game

## Overview
This project is a simple slot machine game built in Unity as part of a technical assignment.  
It demonstrates core concepts like reel animation, RNG, UI interaction, and game logic.

---

## Features
- 3 Reel Slot Machine
- Randomized symbol generation (RNG)
- Smooth reel spinning effect
- Sequential reel stopping
- Win detection system
- Payout system based on symbol match
- Clean UI interaction

---

## Approach
- Implemented a modular system with:
  - `ReelController` for reel behavior
  - `Symbol` and `SymbolView` for data + visuals
  - `UIManager` for game flow and UI handling
- Used object-oriented design for scalability
- Implemented masking for reel window effect

---

## How to Run
1. Open the project in Unity
2. Open `MainScene`
3. Click Play
4. Press the SPIN button

---

## Bonus Improvements
